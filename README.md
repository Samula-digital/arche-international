# Arche' International Rebuild

This folder is the first rebuild workspace for Arche' International.

## Files

- `index.html` - public-facing homepage draft.
- `who-we-are.html` - organization story and partnership model.
- `mission.html` - mission, values, and accountability policy draft.
- `programs.html` - program areas and historical regions.
- `donate.html` - donation readiness page with online giving paused until payment setup.
- `volunteer.html` - volunteer invitation and temporary mail form.
- `transparency.html` - IRS public record and launch-readiness notes.
- `faith-statement.html` - concise faith statement draft.
- `contact.html` - contact page and temporary mail form.
- `assets/css/styles.css` - site styling.
- `assets/js/main.js` - mobile menu and hero image rotation.
- `assets/images/` - archived visual assets from the old site.
- `content/archive-rebuild-notes.md` - source notes and content map from the Wayback research.
