```text

```

Public contact:

```text

```

## After Registering The Domain

1. Add the custom domain in GitHub Pages for `Samula-digital/arche-international`.
2. Add the required DNS records at the domain registrar.
3. Wait for GitHub Pages to verify DNS.
4. Enable HTTPS.
5. Add a `CNAME` file containing:

```text
archeintl.com
```

Do not add the `CNAME` file before the domain is registered and DNS is ready, because it can interrupt the current GitHub Pages URL.
